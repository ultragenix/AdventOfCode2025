# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Python.

## Structure

Each day's solution is organized in its own directory:
```
dayXX/
├── input.txt
├── solution.py
└── README.md
```

## Running Solutions

```bash
python dayXX/solution.py
```
